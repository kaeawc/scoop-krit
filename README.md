# scoop-krit

Scoop bucket for [krit](https://github.com/kaeawc/krit). Manifests under
`bucket/` are auto-published by [GoReleaser](https://goreleaser.com/) on
every tagged `v*` release of `kaeawc/krit`.

## Install

```powershell
scoop bucket add krit https://github.com/kaeawc/scoop-krit
scoop install krit
```

This is intentionally a thin manifest mirror — please file issues
against [`kaeawc/krit`](https://github.com/kaeawc/krit/issues), not
this repo.
